# Erlanger Disciples Redesign

This repository contains a modern, responsive redesign of the Erlanger Disciples Church website built with Bootstrap 5, custom CSS, and vanilla JavaScript.

## Getting Started

Open `index.html` in your browser to explore the site locally. All assets are loaded via CDN or from the `assets` directory, so no build step is required.

## Features

- Hero section with service highlights and call-to-action buttons
- About, events, ministries, beliefs, sermons, visit, and contact sections
- Filterable events grid and ministry detail modals
- Animated scroll reveals and interactive forms powered by JavaScript
- Custom styling layered on top of Bootstrap for a distinct visual identity
